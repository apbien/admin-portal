# admin-portal
To run:
1) Unzip the node_modules folder OR download these dependencies from npm:
	npm i express
	npm i serve-favicon
	npm i cookie-parser
	npm i express-session
	npm i dotenv
	npm i cors
	npm i bcrypt
	npm i sequelize
	npm i mysql2
1) Go to project folder using the command prompt
2) Type 'node start' and enter

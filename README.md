# admin-portal
Requirements:
- mySQL
- Node.js
- npm

To run:
1) Unzip the node_modules folder OR download these dependencies from npm:
	npm i express
	npm i serve-favicon
	npm i cookie-parser
	npm i express-session
	npm i bcrypt
	npm i sequelize
	npm i mysql2
2) Go to project folder using the command prompt
3) Type 'node start' and enter
